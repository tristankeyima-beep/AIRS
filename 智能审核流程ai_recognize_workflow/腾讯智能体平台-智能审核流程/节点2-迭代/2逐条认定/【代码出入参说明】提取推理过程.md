# 代码节点出入参说明：提取推理过程

对应代码：`提取推理过程.py`

## 这个节点做什么

把逐条认定 LLM 的推理内容整理成稳定字符串，输出 `reasoningContent`。

## 入参

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `reasoningContent` | String、Object 或 JSON字符串 | 否 | 推理过程文本。为空时输出空字符串。 |

### 可接受层级

```text
reasoningContent: String
```

或：

```text
reasoningContent: Object
  reasoningContent: String
```

## 出参

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `reasoningContent` | String | 推理过程文本。 |

## 可直接粘贴测试的入参示例

```json
{
  "reasoningContent": "规则要求患者确诊为二型糖尿病，但材料记载无糖尿病史，因此该规则不通过。"
}
```

## 对应出参示例

```json
{
  "reasoningContent": "规则要求患者确诊为二型糖尿病，但材料记载无糖尿病史，因此该规则不通过。"
}
```
