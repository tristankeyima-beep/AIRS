# 代码节点出入参说明：精解结果结构化

对应代码：`代码-将精解结果结构化.py`

## 这个节点做什么

把“材料精解”模型输出整理成数组，并按 `keywordCode` 排序，供后面的逐条认定使用。

## 入参

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `extraction_data` | Array 或 JSON字符串 | 否 | 模型输出的精解结果。可以是数组对象，也可以是包含 JSON 的文本。为空时输出空数组。 |

### `extraction_data` 推荐层级

```text
extraction_data: Array
  - item: Object
      keywordCode: String，提取项编码
      found: Boolean 或 String，是否找到相关材料证据，具体由提示词输出决定
      results: Array 或 Object 或 String，提取到的结果，具体结构由提示词输出决定
```

说明：这个节点不强校验每个 item 的内部字段，只要求最外层是数组；排序时会读取 `keywordCode`。

## 出参

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `extractionList` | Array<Object> | 结构化后的精解结果数组，按 `keywordCode` 升序。 |

### 出参层级

```text
Object
  extractionList: Array
    - item: Object
        keywordCode: String
        其他字段: 任意类型，来自模型输出，原样保留
```
